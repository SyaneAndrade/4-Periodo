Emissora Conexão com o Banco de Dados:

1 - Crie um database chamado Emissora;

2 - Crie todo o banco de dados dentro da database Emissora;

3 - Você precisara do Eclipse instalado, da library postgresql-(versão do seu postgresql) -jdbc.jar para conexão com o banco;

4- Abra seu Eclipse e escolha a pasta onde o projeto em java do DB está salvo;

5 - Add o jdbc.jar a library do seu eclipse;

6- Vá até a aba conexão na aplicação em java e :
		String user= "user"; (Nesse campo insira seu nome de usurio que da acesso ao seu BD)
		String password= "password"(Nesse campo insira sua senha para acesso ao DB);
		
7 - A main eu fiz pra testar todas as conexões e retornos com o banco está correto, testando pra cada entidade adcionada a aplicação em Java.
